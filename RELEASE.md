# Memo Timeline Feed v1.0.0

## Highlights

- Add `Timeline` view for notes tagged with `#timeline`
- Add `Diary Feed` view with a Moments-style card layout
- Add `Photos` mode for image-only browsing
- Add configurable timeline tag, diary tag, diary folder, and daily note filename pattern
- Add polished Obsidian-native UI based on the provided visual reference
